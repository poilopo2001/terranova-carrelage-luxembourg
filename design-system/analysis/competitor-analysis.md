## 6. Design Philosophy

### Analyse par concurrent
- **AlloCarrelage** : Template WordPress custom, orange industriel, orientation service B2C basique. Aucune elevation esthetique.
- **Maroldt** : Corporate sobre, le carrelage est une activite parmi d'autres. Design neutre volontairement.
- **Luxcarrelage** : Template Webador (constructeur de site gratuit). Niveau amateur evident.
- **Domingues** : Template Divi surutilise. Design "agence web locale" reconnaissable. Professionnel mais sans ame.
- **Ekelmann** : WordPress block editor basique. 30 ans d'experience, site qui en parait 10.

### Constat global
**Tous les concurrents sont visuellement interchangeables** : templates generiques, couleurs froides/industrielles, aucune personnalite de marque. Aucun ne communique visuellement le savoir-faire artisanal, la noblesse des materiaux, ou la passion du metier.

### Philosophie Terranova
Terranova doit etre le premier carreleur luxembourgeois dont le site **ressemble a son travail** : precis, chaleureux, noble. La palette matiere (brun/ocre/ivoire) evoque directement les pierres posees. La typographie serif apporte l'elegance artisanale. Le spacing genereux imite la respiration d'un beau sol. Les animations subtiles traduisent la precision du geste.

## 8. Key Learnings for Adaptation

### Ce que TOUS les concurrents font (a eviter/depasser)
1. Templates generiques (WordPress/Divi/Webador) - Terranova : site custom Next.js
2. Couleurs froides/industrielles - Terranova : palette matiere chaude
3. Sans-serif generiques partout - Terranova : serif elegant en titres
4. Pas d'animations - Terranova : micro-interactions premium
5. Spacing inconsistant - Terranova : grille 8dp rigoureuse
6. Cards plates sans elevation - Terranova : shadows, hover effects
7. Hero section basique (slider/image) - Terranova : hero cinematique
8. Formulaires generiques - Terranova : formulaire design avec validation inline

### Differentiation visuelle Terranova
1. **Palette unique** : seul site du secteur en tons terre/ocre/ivoire
2. **Typographie distinctive** : seul a utiliser une serif (Playfair Display)
3. **Niveau technique** : Next.js 14 vs WordPress/Divi/Webador
4. **Animations premium** : scroll-triggered, hover elevations, transitions
5. **Grille rigoureuse** : 8dp system vs spacing anarchique
6. **Accessibilite** : WCAG AA natif vs approximatif
7. **Performance** : Lighthouse 90+ vs 50-70 typique templates

### Inspiration hors-secteur
- **Airbnb** : chaleur, espaces genereux, galerie immersive
- **Apple** : precision, animations subtiles, typographie soignee
- **Cereal Magazine** : editorial, serif elegant, mise en valeur photo
